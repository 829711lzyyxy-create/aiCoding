# OfferPilot - 校招 Offer 决策助手

一个帮助校招候选人快速对比和分析多个工作 Offer 的单页应用。

## ✨ 功能特点

- **薪资测算**：输入城市、月薪、发薪月数、年终奖、公积金比例，自动计算每月到手和年度税后现金
- **多城市口径**：支持上海、北京、深圳、杭州、成都等城市的社保公积金政策
- **智能模板**：内置互联网大厂、外企、制造业研发、国企等典型 Offer 模板
- **多 Offer 对比**：将多个 Offer 放到同一口径下对比，快速看出差异
- **偏好模式**：支持现金流优先、求稳优先、长期成长三种偏好模式
- **AI 分析**：接入大模型进行深度分析，提供专业对比建议
- **报告导出**：支持导出为 PDF、分享链接、下载 JSON 数据

## 🚀 技术栈

- **前端框架**：原生 HTML + CSS + JavaScript
- **数据存储**：LocalStorage 本地存储
- **布局方式**：Flexbox + CSS Grid
- **样式风格**：现代简约设计，响应式布局

## 🎯 使用方法

1. **打开应用**：直接在浏览器中打开 `offer_transparency_interactive_ui_v5.html`
2. **录入 Offer**：选择手动填写、粘贴文本识别或智能模板
3. **添加对比**：将多个 Offer 加入对比集合
4. **AI 分析**：点击 AI 分析按钮获取深度分析报告
5. **导出报告**：点击导出报告按钮生成 PDF
